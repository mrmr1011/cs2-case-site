# CS2 Case Opening Demo

Bu proje, **Counter-Strike 2 (CS2)** temalı bir **kasa açma demo sitesi**dir.  
Gerçek case sitelerine benzeyen özellikler içerir fakat **Steam ile bağlantısı yoktur** ve tüm işlemler **demo amaçlıdır**.

## Özellikler

- 🎰 **Gerçek slot animasyonu** (kayan roulette)
- 🖼 **Gerçek CS2 skin görselleri**
- 📦 **Çoklu kasa sistemi**
- 🎒 **Inventory sistemi** (skinleri saklayabilme)
- 💰 **Skin satma (fake balance)**  
- 🔼 **Upgrade sistemi** (item değerini yükseltme denemesi)
- ⚔ **Case Battle** (bot ile basit battle)
